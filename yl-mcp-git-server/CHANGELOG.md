# 更新日志

## [1.0.1] - 2025-11-12

### 修复
- **重要修复**: 修复通过 npx 调用时 git 命令在错误目录执行的问题
  - 将 `PROJECT_ROOT` 从相对路径改为使用 `process.cwd()` 获取调用者的当前工作目录
  - 现在 git 命令会在用户的项目目录而不是 MCP 服务器安装目录中执行
  - 解决了找不到 `.git` 目录的问题

### 改进
- 更新文档，说明已修复的工作目录问题
- 在故障排除部分添加相关说明

## [1.0.0] - 2025-11-12

### 新功能
- 🚀 初始版本发布
- 🛠️ 支持基本的 Git 操作：init, status, diff, add, smart_commit
- 🤖 智能提交功能，AI 自动分析代码变更并生成规范的 commit 信息
- 📦 支持通过 npx 直接使用，无需安装
- 🔧 支持仓库初始化和远程地址配置
- 📊 提供详细的状态查看和变更分析
